## Technical stack

- Monorepo: Turborepo
- Application: Typescript, Nextjs
- Micro Animation: NextUI
- Styling: TailwindCSS
- Testing: Vitest
- Component Documentation: Storybook
- Package Manager: pnpm

## Getting Started

### 1) Install Dependencies

```shell
pnpm install
```

### 2) Run the Development Server

```shell
pnpm run dev
```
